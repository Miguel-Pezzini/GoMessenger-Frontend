Crie o design de um sistema web de chat entre duas pessoas, inspirado no modelo do WhatsApp Web e do Discord.

O sistema deve ter layout desktop (1440px de largura), moderno, minimalista e focado em usabilidade.

📌 Estrutura geral da interface:
1️⃣ Sidebar esquerda (lista de conversas)

Largura aproximada de 280–320px

Campo de busca no topo com placeholder: "Pesquisar conversa"

Lista vertical de contatos/conversas contendo:

Foto de perfil (avatar circular)

Nome do contato

Última mensagem (preview curta)

Horário da última mensagem

Indicador visual de mensagens não lidas (badge com número)

Destaque visual para conversa selecionada

Scroll vertical suave

Botão no topo para "Nova conversa"

2️⃣ Área principal de chat (centro-direita)

Nome do contato no topo

Status (online, offline, digitando…)

Botões no header:

Chamada de voz

Chamada de vídeo

Menu de opções

3️⃣ Área de mensagens

Mensagens alinhadas:

Direita: mensagens do usuário atual

Esquerda: mensagens da outra pessoa

Balões arredondados

Timestamp pequeno abaixo ou ao lado da mensagem

Suporte visual para:

Mensagens longas

Emojis

Imagens enviadas

Scroll automático para a última mensagem

4️⃣ Input de mensagem (parte inferior)

Campo de texto grande com placeholder: "Digite sua mensagem..."

Botão para anexar arquivo

Botão de emoji

Botão de enviar

Enter envia mensagem

Shift + Enter quebra linha
